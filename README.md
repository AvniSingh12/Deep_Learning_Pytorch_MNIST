# Introduction to Deep Learning using PyTorch — MNIST example

Simple, well-commented PyTorch example of:
- Downloading and loading MNIST
Small feedforward network (possibly with dropout) or small CNN
- Training loop, evaluation, and optional visualization
- GPU detection with automatic CPU fallback

- `Intro_Deep_Learning_PyTorch_MNIST.py` — main script (run from the command line)
